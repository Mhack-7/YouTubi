## 🎛 Що вміє YouTubi

### 🚫 Реклама
| Функція | Опис |
|---|---|
| **Пропуск реклами** | Автоклік «Пропустити», приховування оверлеїв |
| **SponsorBlock** | Автопропуск спонсорів, інтро, аутро через публічне API |
| **Відновлення якості** | Повертає якість відео до обраного після реклами |

### 🎬 Плеєр і відео
| Функція | Опис |
|---|---|
| **Примусова якість** | Завжди відкривати відео у вибраній якості (144p – 8K) |
| **Зациклення** | Кнопка в плеєрі + клавіша `L` |
| **Скриншот** | Кнопка + `Ctrl+Shift+S`, збереження PNG |
| **Залишковий час** | Показ `-HH:MM:SS (XX%)` поруч з тривалістю |
| **Збереження прогресу** | Повернення до місця зупинки після перезавантаження |
| **Примусова гучність** | Фіксована гучність при кожному відео |
| **Примусова швидкість** | Фіксована швидкість відтворення між відео |
| **Зміна гучності** | `Shift + Scroll` — крок 1% |
| **Зміна швидкості** | Скрол або ПКМ на кнопці «Налаштування» |
| **Вимкнути засинання** | Плеєр не ставить паузу при неактивності |

### 💬 Коментарі
| Функція | Опис |
|---|---|
| **Дизлайки** | Повертає лічильник через Return YouTube Dislike API |
| **Точні лайки** | `1.2K` → `1 247` |
| **Сортування** | Авто-вибір «Популярні» або «Нові» при відкритті |
| **Підсвічування автора** | Коментарі власника каналу — з кольоровою рамкою |
| **Згортання відповідей** | Всі відповіді згорнуті за замовчуванням |
| **Переклад** | Кнопка перекладу під кожним коментарем |

### 🖥 Інтерфейс
| Функція | Опис |
|---|---|
| **Перефарбування** | Повна зміна теми YouTube (6 пресетів + власні кольори) |
| **Шрифт Ubuntu** | Заміна стандартного шрифту |
| **Повні назви відео** | Без обрізання в усіх списках |
| **Приховати Shorts** | Прибирає Shorts з головної, панелі, каналів |
| **Приховати трансляції** | Прибирає прем'єри та лайви з головної |
| **Компактна панель** | Бічна панель завжди у мінімальному вигляді |
| **Аватар каналу** | Показує аватар поруч з назвою у рекомендаціях |
| **Плавні анімації** | Fade-in при завантаженні сторінки |
| **Фільтр коротких** | Ховає відео коротше N хвилин на головній |
| **Синя іконка** | Кастомна іконка у вкладці браузера |

### 🔍 Навігація
| Функція | Опис |
|---|---|
| **Копія посилання з часом** | Кнопка в плеєрі — копіює `youtu.be/ID?t=XXX` |
| **Вкладка каналу** | Запам'ятовує останню відкриту вкладку каналу |
| **Виправлення посилань** | Коректні посилання на канал у бічній панелі |
| **Пошук в новій вкладці** | СКМ на кнопці пошуку або підказці |
| **Кількість відео** | Показує кількість відео поруч з підпискою |

### 🎨 Кастомізація
| Функція | Опис |
|---|---|
| **Кольорові теми** | ReTube · ReTube Dark · Темний · Пурпурний · Зелений · Власний |
| **Власний CSS** | Вбудований редактор, застосовується одразу |

## ⌨️ Гарячі клавіші

| Клавіша | Дія |
|---|---|
| `F2` | Відкрити / закрити меню YouTubi |
| `L` | Зациклити відео *(якщо увімкнено)* |
| `Ctrl + Shift + S` | Скриншот кадру *(якщо увімкнено)* |
| `Shift + Scroll` | Гучність ±1% *(якщо увімкнено)* |

> Меню можна **перетягувати** за заголовок `YouTubi`. Позиція зберігається між сесіями.

## 🔧 Як це працює

YouTubi використовує **YouTube Data API v3** (через пул публічних ключів з `yt_api_keys.json`) для:
- отримання дати завантаження відео
- отримання кількості відео на каналі
- показу аватарів каналів у рекомендаціях
- точної кількості лайків

Ключі кешуються в `localStorage`. SponsorBlock використовує публічне API [sponsor.ajay.app](https://sponsor.ajay.app). Дизлайки — [returnyoutubedislike.com](https://returnyoutubedislike.com).
